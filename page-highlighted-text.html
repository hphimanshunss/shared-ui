The highlighted text component is meant to be used to mark or highlight text.

General text here. Links in the notification.

Documentation

Required Markup

To use highlighted text add class sui-highlighted-text, also do not forget to add <mark></mark> tag outside the text which needs to be highlighted.

By default background color of highlighted text will be yellow to modify use background-color: yellow; and to modify the color of text use color: black;.

<!-- Highlighted text -->

<div id="unique-id" class="sui-highlighted-text">
	
	<div class="sui-highlight-content">
		
		<div class="sui-highlight-message">
		
			<p>General text here. <mark style="background-color: red; color: white;">Links in the notification</mark>.</p>
		
		</div>
	
	</div>

</div>
